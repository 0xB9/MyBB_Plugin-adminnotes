Plugin Description:

Administrator notes saves and displays them in a list.

This list has pagination of notes!
Notes can be deleted from a table that appears below the textarea and can only remove or empty the administrators!

Install:
Upload the plugin folder to inc/plugins and the index.php file to admin/modules/home after installing the plugin, if you want to have a configuration for the paging of notes!

If the plugin is disabled the notes are saved as before the plugin is not necessary to restore the file index.php
